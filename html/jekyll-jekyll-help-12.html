afeld Saturday, October 04, 2014

I'm looking for the same functionality. It's a shame that, for the http://jekyllrb.com use case (at least), the order is essentially specified in three places:

  • In the data file
  • As the prev_section in the page metadata
  • As the next_section in the page metadata

Only using the data file would be nice because it has the ability to group the pages and you can see the order at a glance, but it would be messy to then create the section_nav from the site.data.docs. jekyll-page-collections seems like it solves this problem, but of course, no Pages support :wink:
