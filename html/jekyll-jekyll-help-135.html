AndyCErnst Saturday, August 23, 2014

Ah, finally figured it out. The parts I feel should really be mentioned on the site/article are: _config.yml file must have:

excludes: ["vendor"]

The Procfile must have:

web: jekyll serve -P $PORT

There has to be a Gemfile with at least:

source "https://rubygems.org"
ruby "2.1.1"
gem "rack-jekyll", "~> 0.4.1"
