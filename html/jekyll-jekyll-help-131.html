How can I specify multiple types? (e.g. setting the same value for both posts and drafts)

I’m not sure if the type drafts is even valid. Drafts are just posts that are not published (and not even generated by Jekyll). Front matter defaults won’t have any effect on them.

I also wanted to set some values using wildcards such as .svg, or index.html on any directory, or foobar/ (any file inside foobar/).

Also path: foobar matches every thing Jekyll recognizes as a document. However I found myself respecifying the same values with path: foobar/_posts to get it to work with posts in a subdirectoy.

How is this possible? You either have /about/index.html or /about.html. If you have both, you probably have two different pages for these.
