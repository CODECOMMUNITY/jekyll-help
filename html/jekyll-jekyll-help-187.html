The {{ double brackets }} don't work because the Liquid template uses them.

troyswanson Thursday, November 06, 2014

You can use the {% raw %} tag to prevent Liquid from parsing stuff. Like this:

<span>You have {% raw %}{{ unread_count }}{% endraw %} unread messages</span>
