kleinfreund Friday, July 18, 2014

Have a look at my categories.html:

{% for category in site.categories %}
    {% assign posts = category[1] | where:"lang","en" %}

    {% if posts.size > 0 %}
    <h2 id="{{ category[0] }}">{{ category[0] }}</h2>
    <ul>
        {% for post in posts %}
        <li>
            <a href="{{ post.url }}">{{ post.title }}</a>
        </li>
        {% endfor %}
    </ul>
    {% endif %}

    {% assign posts = nil %}
{% endfor %}

It lists all categories and English posts in them. However you just need a smaller bit:

<ul class="category-list">
{% for category in site.categories %}
    <li>{{ category[0] }}</li>
{% endfor %}
</ul>
