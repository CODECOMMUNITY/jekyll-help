Get excerpt from specific element

winghouchan Monday, September 08, 2014

Howdy peeps!

I'd like to ask if it is possible to get an excerpt from a specific element?

For example, let's say I have a blog post and in it is:

<div id="excerpt">
  <!-- Excerpt -->
</div>

I then want the content, HTML and all, within said <div> to be used as the excerpt in post list.

kleinfreund Tuesday, September 09, 2014

That is not possible at the moment. However, you can set excerpt in the front matter and put the content there.

---
excerpt: "Whatever you want it to be…"
---
winghouchan Tuesday, September 09, 2014

I need the HTML within the <div> to be parsed as part of the excerpt as well.

kleinfreund Tuesday, September 09, 2014 
---
excerpt: "<div class='note'><p>Whatever you want it to be…</p></div>"
---

…should do the trick. Be careful with " and '.
