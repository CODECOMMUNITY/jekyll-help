Using a partial in a markdown file - should I need to wrap it in HTML?

barryclark Monday, November 24, 2014

When including an HTML partial in the middle of a markdown file it seems that I need to wrap it in a div. Am I doing something wrong?

This doesn't work, but I'd expect it to:

{% include call-to-action.html %}

This does work, but I need to wrap it up in some HTML:

<div>{% include call-to-action.html %}</div>

troyswanson Monday, November 24, 2014

You don't have to. It really depends on what's in call-to-action.html. All the {% include ... %} tag does is literally include the contents of the file into the layout.

Some more context would be helpful if you need some help.
