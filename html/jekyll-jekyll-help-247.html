How can I limit results to one match when two conditions match?

tomjohnson1492 Thursday, January 22, 2015

I want to cycle through my pages and get all pages that have either a tag called "release_notes" or a tag called "news." Here's my code:

{% for page in site.pages limit: 5 %}
{% for tag in page.tags %}
{% if tag == "release_notes" or tag == "news" %}
<li><a href="{{ page.permalink | prepend: site.baseurl }}">{{page.title}}</a></li>
<div class="summary">{{page.summary}}</div>
{% endif %}
{% endfor %}
{% endfor %}

One of my pages has front matter as follows:

---
layout: page
title: 2.0 Release Notes
permalink: /release_notes_2_0/
tags: 
- release_notes
- news
---

This page appears twice in the for loop. How can I make it so that the page only appears once?

(I realize the simple solution would be to restrict the tags to one per page, or restrict the for loop to one tag per loop, but maybe it's easier to do this than I thought.)

    {% endunless %}{% endfor %}
