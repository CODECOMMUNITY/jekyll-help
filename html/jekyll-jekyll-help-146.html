janvanmansum Tuesday, October 14, 2014

The one from biosphere works for me.

  • Copy-paste the plugin code to <your-project>/_plugins/breadcrumbs.rb

  • Create a file <your-project>/_includes/breadcrumbs.html:

         <ol class="breadcrumbs">
         {% for p in page.ancestors %}
         <li><a href="{{ site.baseurl}}{{ p.url }}">{{ p.title }}</a></li>
         {% endfor %}
         <li class="active">{{ page.title }}</li>
     </ol>

  • Create a file <your-project>_sass/_breadcrumbs.scss:

      .breadcrumbs {
      margin-left: 0;
      li {
             display: inline;
             list-style: none;
             &:after {
                content: " > ";
             }
            &.active {
                &:after {
                    content: "";
            }
         }
      }
  }
  • Add the _breadcrumbs.scss to <your project>/css/main.scss:
      @import
  "base",
  "layout",
  "syntax-highlighting",
  "breadcrumbs",
  ....
    Then the titles of the index files show up something like this:

Home > Manual > Code Style

if your directory tree is laid out like this:

 [root of project] 
        - index.html  (with "title: Home" in the metadata)
        [manual]
             - index.md (with "title: Manual"  in the metadata)
             [code]
                   - index.md (with "title: Code Style" in the metadata)

For posts too many ">" characters appear though.
