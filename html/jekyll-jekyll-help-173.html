How can I include a transformed to HTML markdown file?

aleksey-bykov Sunday, October 19, 2014

I use {% include something.md %} and jekyll does it literally without transforming md to html. How do I enable the transformation?

jaybe-jekyll Sunday, October 19, 2014

You may capture and filter through Markdown.

{% capture something %] {% include something.md %} {% endcapture %}

{{ something | markdownify }}

http://jekyllrb.com/docs/templates/#filters

troyswanson Sunday, November 02, 2014

Usually, includes are already HTML (think of them as modules for the layout). However, @jaybe-jekyll's solution should work as advertised.
