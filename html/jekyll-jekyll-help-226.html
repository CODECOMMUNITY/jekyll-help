troyswanson Sunday, January 04, 2015

Gents, I would highly encourage you to not use baseurl and instead use GitHub Pages Metadata.

A quick example: Assuming you have a CSS file called /css/styles.css relative to the root of your repo, you can use something like <link rel="stylesheet" href="{{ site.github.url }}/css/styles.css">. The {{ site.github.url }} value will be replaced by the URL for your repo's site.
