alfredxing Sunday, December 28, 2014

You'll need to upgrade your Ruby to version 1.9.3 or above, though I'd recommend the latest stable, 2.2.

Since you're on OS X, I would recommend doing this by using a Ruby manager. Some good choices are:

  • RVM
  • rbenv with the ruby-build plugin
  • chruby with the ruby-install tool

If you're only going to use a single Ruby, Homebrew is also a good option.

You can also upgrade your OS to Mavericks (10.9) or Yosemite (10.10); these should include a more up-to-date version of Ruby.
