troyswanson Friday, September 05, 2014

@parkr Is on to something there. You might also try removing the exclude: [] line from your _config.yml file.

PS: You don't have to specify a date string in your post's front-matter like that. Liquid has all kinds of fancy date filters that allow you to format the date that is automatically generated via the file name.

For example, if you have a post that has a file name of 2014-09-01-Making-me-mad.md, Jekyll will automatically send a date value into Liquid for you to format. The format that you indicated in your example would look like this in your layout: {{ page.date | date: "%B %d, %Y" }}
