Looks like relative includes already are supported in the gh-pages gem (thanks again @gjtorikian).

It works via gh-pages v28. :+1:

For example, here's site.coffee:

---
---

{% include_relative _partials/foo.js %}
{% include_relative _partials/bar.js %}

number = 42

And here's foo.js:

console.log 'foo'

… and bar.js:

console.log 'bar'

Here's the combined and preprocessed site.js:

(function() {
  var number;

  console.log('foo');

  console.log('bar');

  number = 42;

}).call(this);
