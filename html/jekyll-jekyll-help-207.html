redknitin Saturday, December 06, 2014

@FabianCernota You are probably just missing the Ruby Dev package. You can install it on Ubuntu or other Debian-based Linux by typing in a terminal window: sudo apt-get install ruby-dev . On a Fedora/RedHat-based Linux, it would be a sudo yum install ruby-devel

After installing Ruby Dev, attempt to install the Jekyll gem again.
