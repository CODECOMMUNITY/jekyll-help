It seems that you need to fix some permissions first. Go to Disk Utility, select your Macintosh HD and Press Repair Permissions. Lets see if we can get rid of warning: Insecure world writable dir /usr/local in PATH, mode 040777. After that, try to run xcode-select --install. This command should pop up something like a banner that you have to press install to get the command line tools. This will not install xcode though. Remember that you can also download the latest Command Line Tools from the Developer Apple site. BTW, if you type brew doctor what's the output?
