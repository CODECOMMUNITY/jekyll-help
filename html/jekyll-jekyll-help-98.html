parkr Thursday, July 17, 2014

I meant in your config. It looks like we've locked down import paths. We chose to go with Sass rather than Compass's ruby libraries for Sass's simplicity. I think we can open up import paths when not in safe mode. This would allow you to do:

sass:
  sass_dir: _sass
  load_paths:
    - "C:/develop/Ruby200-x64/lib/ruby/gems/2.0.0/gems/compass-0.12.6/frameworks/compass/stylesheets/"

But it's not currently available.
