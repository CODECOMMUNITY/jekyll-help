albertogg Sunday, May 25, 2014

The easiest way is to match what your development jekyll with GitHub Pages jekyll is to use the github-pages gem. AFAIK the documentation is not versioned so searching only the 1.5.1 docs will not be posible, although I'm not sure. You can take a look at the changelog here 

$ gem install github-pages

or add it to your Gemfile, the latest version is 19.

gem 'github-pages', '~> 19'

Additionally, front-matter defaults are a Jekyll 2.0 feature, and by going to https://pages.github.com/versions/, you would see that GitHub Pages (and the github-pages gem) have not been updated to Jekyll 2.0. This means front-matter defaults are not available on GitHub Pages (yet).
