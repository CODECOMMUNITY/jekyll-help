Hi! I just set up a custom workspace on c9, and managed to get Jekyll working from there:

  1. In Terminal, install Jekyll with gem install jekyll
  2. cd to the directory that contains your Jekyll site
  3. Run jekyll serve --port $PORT
  4. Visit your site at http://<workspacename>-c9-<username>.c9.io
